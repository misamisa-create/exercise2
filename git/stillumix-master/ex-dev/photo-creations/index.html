Creator's Note

7/8(金)より7日間、16人のクリエイターがSTILLUMIX写真展に向けて連日STILLUMIX写真展用のnoteを投稿していく企画となっております。
クリエイターは5枚の組写真を1つのテーマで制作し、
そのうち4枚及び制作意図や制作ストーリーをnoteに掲載。
STILLUMIX写真展での展示を見たときに組写真が揃い、完成します。
クリエイターのnoteはこちらからご確認いただけます。
